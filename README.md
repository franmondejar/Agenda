# 🚀 Agenda Flash PRO

**Agenda Flash** es una aplicación web progresiva (PWA) diseñada para gestionar recordatorios de forma rápida y visual. Desarrollada con un enfoque en la simplicidad y la eficiencia, permite organizar tareas diarias con un sistema de colores intuitivo.

## ✨ Características principales

* 📅 **Gestión de Citas:** Añade recordatorios con título y fecha/hora exacta.
* 🔍 **Búsqueda Dual:** Filtra tus tareas por nombre o por una fecha específica mediante el calendario integrado.
* ⚡ **Accesos Flash:** Botones de acceso rápido para programar tareas a +1 día, +1 semana o +1 mes.
* 🎨 **Interfaz Visual Limpia:** * **Ribete Azul:** Tareas pendientes en plazo.
    * **Ribete Rojo:** Tareas que ya han cumplido su fecha.
* 📱 **Modo PWA:** Instalable en dispositivos móviles como una aplicación nativa.
* 💾 **Persistencia Local:** Tus datos se guardan en el navegador (LocalStorage), sin necesidad de registros ni bases de datos externas.

## 🛠️ Instalación en el móvil

Para llevar tu agenda siempre contigo:
1. Abre el enlace de la web en el navegador de tu móvil.
2. Selecciona **"Añadir a la pantalla de inicio"** o **"Instalar aplicación"**.
3. ¡Listo! Ya tienes el icono en tu escritorio.
